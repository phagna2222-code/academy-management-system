## Data Hygiene вҖ” бһўбһ“бһ¶бһҳбҹҗбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

> вң… **Do вҖ” бһҸбҹ’бһҡбһјбһң**
> - бһ”бҹ’бһҡбһҫ code бһҜбһҖбһҹбһҺбҹ’бһӢбһ¶бһ“ (uppercase, prefix per module)
> - бһҡбһҖбҹ’бһҹбһ¶ Is Main вң“ бһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҖбҹ’бһ“бһ»бһ„бһҹбһ¶бһӣбһ¶
> - бһҡбһҖбҹ’бһҹбһ¶ Is Current вң“ бһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһҖбҹ’бһ“бһ»бһ„ Academy Year
> - Soft-delete (Inactive status) бһҮбҹҶбһ“бһҪбһҹбһҖбһ¶бһҡбһӣбһ»бһ”бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ
> - бһ•бҹ’бһҸбһӣбҹӢ Role бһҷбҹүбһ¶бһ„бһҸбһ·бһ… бҹЎ бһұбҹ’бһҷ user бһҗбҹ’бһҳбһё (бһҳбһ·бһ“бһҳбҹӮбһ“ Super Admin)

> вҡ пёҸ **Don't вҖ” бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһң**
> - бһҖбһ»бҹҶ delete `super-admin` role
> - бһҖбһ»бҹҶ assign `super-admin` role бһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ
> - бһҖбһ»бҹҶ delete user бһҠбҹӮбһӣбһҮбһ¶бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ¶бһҒбһ¶ вҖ” re-assign бһҮбһ¶бһҳбһ»бһ“
> - бһҖбһ»бҹҶбһ”бҹ’бһҡбһҫ password `password` бһ“бҹ… production
